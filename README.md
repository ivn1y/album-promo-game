# album-promo-game

Короткая браузерная point-and-click игра в готическом пиксельном стиле, создаваемая как промо-экспириенс для музыкального альбома.

## О проекте

Игра представляет собой серию коротких атмосферных глав, где каждая глава связана с отдельным треком альбома. Игрок взаимодействует с небольшими локациями, персонажами и объектами, а в конце каждой главы переходит в сюрреалистичную ночную мини-игру.

## Стек

- TypeScript
- Node.js
- Phaser
- Vite

## Статус

Сейчас проект находится на этапе vertical slice.
Текущий фокус — первый уровень: прибытие героя в город, первая point-and-click сцена, ночлег и первый переход в мини-игру.
